### Tech Stack Used
- Axios - to fetch the data
- Redux Thunk - Middleware
- React Routing
- ES6 Features
- React Dev Tool extensions to Visualize the store data
- React-Redux Store implementation using Reducers and Actions
- Mixed of Functional and Class Component
